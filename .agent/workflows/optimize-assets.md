description: How to optimize 3D assets (GLB/Textures)
---
// turbo
1. Optimize GLB meshes:
   `npm run assets:gltf:optimize`

2. Convert textures to KTX2 (GPU-ready format):
   `npm run assets:convert-solar` 
   or 
   `npm run assets:convert-nasa`

3. Inspect asset size/structure:
   `npm run assets:inspect`
