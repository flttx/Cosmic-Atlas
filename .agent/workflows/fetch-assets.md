---
description: How to fetch celestial assets (textures/models) for the project
---
// turbo-all
1. Fetch core assets:
   `npm run assets:fetch`
2. Fetch NASA specific high-res models:
   `npm run assets:fetch-models`
3. Fetch Solar System textures:
   `npm run assets:fetch-solar`
4. Setup basis/draco decoders:
   `npm run assets:setup`
   `npm run assets:copy-draco`
